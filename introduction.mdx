The MCP protocol opened a door. AI models can now discover and call tools in a standardised way. But building an MCP server for every database is tedious. Testing those servers for security is an afterthought. And connecting everything takes longer than it should.

0cmplx fixes this. Connect your database, and the engine turns it into a queryable MCP server, instantly. Your tables become tools. Your relationships become intelligence. Your queries execute in a secure sandbox.

## Try it now

No sign-up required. Pick a sample database and start querying:

```bash
npm install -g @0cmplx/cli
0cmplx try banking
```

Or try from the [dashboard](https://0cmplx.com) with one click.

## Three ways in

<Steps>
  <Step title="Try with a sample">
    Pick a pre-built database (banking, ecommerce, healthcare). Explore tables, query in natural language, run security traps. No files, no credentials.
  </Step>
  <Step title="Bring your schema">
    Upload a SQL schema file. The engine parses your tables and relationships. Test with sandbox data before going live.
  </Step>
  <Step title="Connect your database">
    Provide database credentials. Same tools, real data. The schema you tested with stays the same.
  </Step>
</Steps>

## How it works

1. **The engine understands your data.** Tables, columns, foreign keys, constraints. It builds a relationship graph.
2. **Natural language to SQL.** Ask a question. The engine composes the right query, joining across tables automatically.
3. **Sandboxed execution.** Every query runs in isolation. No access to production data. Safe by default.
4. **Security testing.** OWASP traps probe any MCP server for injection, data exposure, broken access, and resource exhaustion.
